# Dashboard de Finanças
Um dashboard moderno e intuitivo para controle financeiro pessoal, desenvolvido com TypeScript e vanilla JavaScript.

## Funcionalidades

- **Adicionar transações** de entrada e saída
- **Visualização em tempo real** do resumo (Entradas, Saídas e Saldo)
- **Filtros** por tipo (Todas / Entradas / Saídas)
- **Categorização** completa (Alimentação, Transporte, Moradia, Saúde, etc.)
- **Design responsivo** (funciona bem no celular)
- **Interface limpa e moderna** com tema dark
- **Dados persistidos apenas na sessão** (fácil de adaptar para localStorage)

- **Vanilla JavaScript** (sem frameworks)
- **HTML5 + CSS3**
